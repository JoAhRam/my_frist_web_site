TEG와 속성 설명


저장 키설정은 Ctrl + s를 누르면 됨
strong과 u 같은 친구들을 TAG라고 하며 150개 이상 TAG라는 게 있다.
strong : 진한 글씨체 태그
u : 밑 줄 태그
h1~6 : '제목 tag' = 줄바꿈, 진한 글씨, 글자 크기가 된다. 1이 제일 높고 6이 제일 낮은 글자 크기다.
br : 줄 바꿈 태그 = 열리는 태그만 붙이면 되는 것이며 닫히는 태그는 필요 없다, 여러번 한 번에 사용 가능하고 두번 이상 쓰면 단락처럼 사용이 가능하다.
p : 단락을 표현 태그 열리는 태그와 닫히는 태그가 필요하다
img : 사진 사용 태그가 필요하다. 웹 폴더 안에 있어야 하며 이미지 tag에는 속성이 필요하다.
이미지의 속성은 src="ㅁ.jpg"라는 이미지의 주소값과 이미지의 사진 크기(픽셀)을 조정 할 수 있는 width="100%"가 필요하다. EX) img src="ㅁ.jpg" width="100%"
목차를 만들고 싶을 시 li를 사용하자. li는 ul이라는 부모태그가 존재하므로 끝과 끝에 붙여주면 깔끔하게 정리가 된다.
BUT 목차부분에 숫자를 넣어야 한다면 ul말고 ol를 사용하면 자동적으로 숫자가 생성된다.
title : 웹의 이름을 변경
meta charset="utf-8" : 한글을 적었을 시 글자가 깨질 수 있으니 넣어주면 됨
title과 meta charset 같은 것은 제목을 지칭하므로 head라는 태그를 사용해야하며 본문은 body를 사용해야한다. 그리고 관용적으로 !dOCTYPE html을 맨 위에 붙여줘야한다.
a 태그는 링크를 붙이고 싶을 때 사용되는 태그이다. a도 속성값이 필요하다. href="주소값"을 넣어주면 된다. 새로운 탭으로 열고 싶다면 target="_blank"를 붙이면 되는 것이고 마우스를 갖다 댔을 때 설명을 하려고 하면 title="내용" EX) a href="ㅁ" target="_blank" title="내용"하면 된다.
